# EraLash Combat — Visual 3.0 Premium Art Direction

Этот пакет усиливает визуал игры без изменения Supabase-схемы.

## Что улучшено

- Новый premium CSS для меню, HUD, кнопок и панелей.
- Кинематографичная арена: глубина, архитектура, световые лучи, туман, перспективный пол, руны.
- Дорогие силуэты бойцов: броня, шлем, плечи, плащ, визор, толстые формы вместо stick-figure.
- Улучшенные спецэффекты атаки: energy arc, slash trail, ultimate diagonal strike.
- Более читаемый бой на телефоне: nameplate под бойцами, контрастные формы, компактный HUD.
- Сохранены все текущие API, Supabase, Stars, missions, boss rush и LiveOps.

## Как установить

1. Распаковать архив.
2. В GitHub Desktop открыть `eralash-combat1`.
3. `Repository → Show in Explorer`.
4. Удалить старую папку `api`.
5. Перетащить все файлы из архива в проект с заменой.
6. Commit:
   `Add Visual 3.0 premium art direction`
7. Push origin.
8. В Vercel дождаться `Ready`.

## Supabase

SQL выполнять не нужно — база данных не менялась.
