# EraLash Combat — Visual 4.2 AAA Presentation Rebuild

Этот пакет исправляет проблему “дешёвого” внешнего вида главного экрана.

## Что изменено

- Главный экран перестроен из узкой технической панели в широкий premium hero-screen.
- Добавлен cinematic showcase справа: VS-frame, 2 бойца, перспектива пола, руны, glow.
- Кнопки систем сгруппированы в Command Deck, меньше ощущения хаотичной админ-панели.
- CTA “Начать бой” стал визуально главным.
- Улучшены типографика, композиция, отступы, фон, свечение и карточки.
- На мобильном сохранён скролл и Telegram-safe layout.
- Backend, Supabase, API, Stars, Admin, LiveOps, Missions, Boss Rush не менялись.

## Деплой

1. Удалить старую папку `api`.
2. Скопировать файлы архива в проект.
3. Commit: `Add Visual 4.2 AAA presentation rebuild`
4. Push origin.
5. Ждать Ready в Vercel.

Supabase SQL выполнять не нужно.
