# Arena Select 15.0 — Premium Stage Selection

Добавлено:
- premium arena select после выбора персонажа;
- 4 арены: Black Citadel, Infernal Bridge, Moon Ritual, Frozen Throne;
- сохранение выбранной арены в localStorage;
- синхронизация арены с VS-screen и боевым HUD;
- уникальный color grading, частицы и атмосфера для каждой арены;
- сохранены Ultimate / Finisher 13.0, AI + Combos 12.0, Combat Physics 11.0 и Combat Feel 10.0.

Проверка:
1. Нажать Начать бой.
2. Выбрать бойца.
3. Нажать Start Fight.
4. Выбрать арену.
5. Нажать Start Fight.
6. В бою в debug/VS-screen должна отображаться Arena Select 15.0 и выбранная арена.
