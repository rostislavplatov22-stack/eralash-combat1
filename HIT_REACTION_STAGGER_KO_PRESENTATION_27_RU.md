# EraLash Combat — Hit Reaction, Stagger & KO Presentation Upgrade 27.0

## Что сделано

- Добавлена система визуальных hit reactions для light / heavy / special / ultimate.
- Добавлен отдельный guard impact для блоков.
- Добавлен stagger state presentation для тяжёлых попаданий.
- Добавлен guard crack feedback при низкой выносливости блока.
- Добавлена более читаемая реакция тела через lean / recoil / небольшое смещение.
- Усилен contact feel: короткий hit-stop, micro camera punch, controlled slow-motion.
- Добавлена KO presentation: cinematic K.O. beat, победная aura-поза, defeat/KO pose проигравшего.
- Добавлен clean combat beat overlay для HIT / STAGGER / BLOCK / K.O.
- Pushback и contact spacing из 26.10 сохранены.
- Facing lock 26.9 сохранён.
- Fighter scale / grounding / idle presence 26.8 сохранены.
- Premium fighter base 26.7 сохранён.
- Start Button Routing 26.6 сохранён.

## Зачем

26.10 сделал бой читаемым, но момент контакта всё ещё ощущался как уменьшение HP.
27.0 добавляет физику реакции: удар попал → тело отреагировало → камера дала импакт → появился stagger / guard impact → раунд заканчивается K.O. beat.

## Acceptance

- Персонажи не просто теряют HP, а визуально принимают удар.
- Блок даёт короткий shield impact без закрытия всего Warden.
- Heavy / special / ultimate дают stagger и более дорогой impact.
- KO выглядит как событие, а не просто переход на result.
- Бой остаётся стабильным и не ломает 26.10.
