# 28.4 Character Select Viewport Fit & Ghlum Premium Polish

## Цель
Исправить рабочий, но визуально ещё тяжёлый Character Select 28.3: убрать обрезание правой панели, сделать Start Fight / Back полностью видимыми на 1080p, уменьшить вертикальную перегрузку и придать Ghlum более дорогую poison-презентацию.

## Что изменено
- Header Character Select стал компактнее.
- Карточки Raven / Iron Warden / Ghlum уменьшены и лучше помещаются в viewport.
- Правая command panel теперь компактнее и не должна обрезать AI Difficulty / Start Fight / Back.
- Уменьшены VS-preview, info-панели, difficulty-кнопки и actions.
- Ghlum получил усиленный toxic premium treatment:
  - зелёный selected glow;
  - улучшенный фон карточки;
  - мягкий toxic bloom;
  - усиленное отделение от тёмного фона;
  - более аккуратную типографику.
- Три реальные карточки из 28.3 сохранены.
- Click-зоны Raven / Iron Warden / Ghlum сохранены.
- Ghlum combat / poison / low-profile логика сохранены.

## Что не трогалось
- Start Fight routing.
- Arena Select.
- Combat 28.0.
- Poison mechanic.
- Hit Reaction / KO 27.0.
- Facing / readability фиксы 26.x.

## Acceptance
- Character Select помещается в 1080p без обрезания правой панели.
- Start Fight и Back видимы.
- Ghlum selectable.
- Выбранный Ghlum читается как premium poison fighter.
- Нет старых reference overlays, fake rails и click-traps.
