# Dark Cinematic Clickmap Fix

Исправление делает кнопки painted-меню активными через дополнительную координатную click-map систему.

Что исправлено:
- большая кнопка «Начать бой» запускает бой;
- левое меню открывает нужные разделы;
- правые карточки Season / Daily / Missions / Stars активны;
- нижние иконки Inventory / Profile / Achievements / Referrals / Settings активны;
- работает даже если прозрачные overlay-кнопки блокируются браузером.

Supabase SQL не нужен.
