# Character Select Luxury Rebuild 27.1

Исправление после визуального ревью: старый Character Select выглядел дешево и оставался на уровне 26.0.

## Что изменено
- Перестроена визуальная подача Character Select в premium luxury стиле.
- Обновлён заголовок версии на `Character Select Luxury Rebuild · 27.1`.
- Увеличены fighter cards, усилена сцена под ногами и rim glow.
- Улучшена презентация Raven и Iron Warden.
- Preview-панель стала чище и дороже.
- Start Fight / Back / Difficulty сохранены без изменения логики.
- 27.0 combat system сохранён.

## Цель
Character Select должен ощущаться как экран дорогой коммерческой игры, а не как старый технический popup.
