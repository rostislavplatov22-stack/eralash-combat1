# 27.3 Prestige Character Select Reference Rebuild

## Что исправлено

- 27.2 выглядел слишком грубо: крупный заголовок залезал на персонажей.
- Raven и Iron Warden были плохо сбалансированы по масштабу и позиции.
- Текст имён слишком сильно спорил с hero-art.
- Правая command panel выглядела перегруженно и слишком узко.
- Нижний portrait rail был слишком шумным.

## Что сделано

- Пересобрана композиция Character Select в более дорогой reference-driven layout.
- Заголовок стал ниже по визуальному весу и больше не ломает hero presentation.
- Нормализованы масштаб и посадка Raven / Iron Warden.
- Имена бойцов переведены в более аккуратную нижнюю hero-инфографику.
- Правая prestige command panel стала плотнее, чище и статуснее.
- Start Fight сохранён как luxury gold CTA.
- Логика выбора бойца, difficulty, Back и Start Fight сохранена.
