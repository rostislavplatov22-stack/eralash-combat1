# 28.3 — Real Fighter Select Rebuild

## Что исправлено
- Полностью отключён сломанный reference-overlay подход 27.5 / 28.1 / 28.2.
- Убраны старые фоновые референсы, fake portrait rail, locked slots и click-trap зоны.
- Экран выбора теперь построен как реальный рабочий UI: 3 большие карточки Raven / Iron Warden / Ghlum + command panel.
- Карточки являются настоящими кликабельными элементами, а не накладками поверх картинки.
- Selected-state теперь привязан к реальной карточке.
- Start Fight запускает выбранного бойца.
- Ghlum, poison и combat flow сохранены.

## Почему это исправление
Предыдущие версии пытались совместить статичный reference art и живой UI. Это создавало визуальный мусор и ломало клики. 28.3 убирает эту архитектурную ошибку и возвращает контролируемый DOM/UI.
