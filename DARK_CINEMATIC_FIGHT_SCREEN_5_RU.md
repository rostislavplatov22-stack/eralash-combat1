# Dark Cinematic Fight Screen 5.0

Обновление поднимает боевой экран до визуального уровня нового Dark Cinematic lobby.

## Что добавлено

- новая dark fortress combat arena на canvas;
- глубокий фон: крепость, башни, луна/адский свет, факелы, туман;
- wet obsidian floor с перспективной сеткой и отражениями;
- runic combat circle в центре арены;
- ember particles и cinematic vignette;
- более дорогой HUD в стиле dark fantasy UI;
- VS intro перед стартом боя;
- плавный переход menu → VS → fight;
- улучшенные мобильные и desktop controls;
- premium victory/defeat result panel.

## Что не менялось

- Supabase schema;
- Telegram Bot API;
- Stars;
- Admin;
- Missions;
- Boss Rush;
- Analytics;
- single Vercel Hobby API route.

## Деплой

1. Удали старую папку `api`.
2. Скопируй все файлы из этого архива в проект.
3. Commit: `Add Dark Cinematic Fight Screen 5.0`
4. Push origin.
5. Дождись Vercel Ready.

Supabase SQL выполнять не нужно.
