# 28.1 Clean Fighter Select & Ghlum Click Hotfix

## Почему
28.0 добавил Ghlum, но Character Select стал перегруженным: Ghlum был вставлен в уже заполненный reference-арт, а реальные click-зоны не совпадали с видимыми элементами. Из-за этого игрок визуально видел Ghlum, но не мог нормально выбрать его.

## Что исправлено
- Убрана перегруженная версия фона `character-select-ghlum-added-28-0.png` из активного Character Select.
- Возвращён чистый approved reference background.
- Добавлен чистый видимый bottom roster dock только на 3 бойцов:
  - Raven
  - Iron Warden
  - Ghlum
- Убраны лишние locked-слоты из активного выбора.
- Отключены невидимые hero-card зоны, которые могли перехватывать клики.
- Добавлены реальные clickable зоны на видимых roster-картах.
- Raven / Iron Warden / Ghlum теперь выбираются через понятные активные кнопки.
- Ghlum получил зелёный selected-state и NEW badge.
- Start Fight / Back / AI Difficulty сохранены.
- Combat 28.0 и poison-механика сохранены.
