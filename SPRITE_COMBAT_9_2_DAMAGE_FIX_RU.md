# Sprite Combat 9.2 — Damage / Hitbox Fix

Исправление после версии 9.1, где спрайты стали видимыми, но удары могли не наносить урон.

Что исправлено:

- атака теперь получает `total = startup + active + recovery`, поэтому state `attack` корректно заканчивается;
- hitbox снова связан с active frames;
- добавлена резервная проверка попадания по центрам бойцов, чтобы удар не промахивался из-за несовпадения sprite sheet и hurtbox;
- увеличена рабочая зона light / heavy / special / ultimate под больших premium-бойцов;
- добавлена метка `SPRITE COMBAT 9.2`;
- HUD получает обновление после каждого попадания.

Проверка:
1. Запусти бой.
2. Дождись надписи `FIGHT`.
3. Подойди к Iron Warden.
4. Нажми `L` или кнопку `L`.
5. HP противника должен уменьшиться.
6. При попадании должны быть hit spark, shake, knockback и floating damage text.

Supabase SQL не нужен.
