# Восстановленный архив EraLash Combat

Это рабочий архив игры для загрузки в GitHub и деплоя на Vercel.

## Что делать
1. Распакуй архив.
2. Загрузи файлы из папки в новый GitHub-репозиторий.
3. В Vercel нажми Add New → Project.
4. Выбери репозиторий.
5. Настройки:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
6. Нажми Deploy.

## Важно
Если Vercel после деплоя просит вход:
Project → Settings → Deployment Protection → выключить Require Log In.
