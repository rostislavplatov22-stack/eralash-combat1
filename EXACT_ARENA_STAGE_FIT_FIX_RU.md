# Exact Arena Stage Fit Fix 5.2

Исправление для точной Black Citadel арены.

## Что исправлено

- Точная картинка арены больше не растягивается на ультраширокий браузерный viewport.
- Арена закреплена как 16:9 stage plate.
- По краям добавлена мягкая cinematic-маска, чтобы не было огромных красных/битых панелей.
- Камера больше не двигает и не зумит фон.
- Бойцы появляются ближе к центральной боевой площадке.
- Движение бойцов ограничено чистой combat zone, чтобы они не уходили в левую декоративную часть карты.
- HUD, Telegram, Supabase, API и экономика не менялись.

## Проверка

После деплоя проверить:

1. Начать бой.
2. Арена должна выглядеть как цельный референс, а не как сдвинутая картинка.
3. Слева не должно быть огромной красной технической панели.
4. Бойцы должны стоять на центральной площадке.
5. Фон не должен ездить при ударах.
