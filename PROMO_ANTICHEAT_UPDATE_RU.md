# Premium Update — Promo Codes + Anti-cheat

## Promo Codes

Игрок может активировать промокод в Mini App через кнопку **🎟️ Promo**.

Дефолтные промокоды:

```text
WELCOME100 — +100 XP / +100 coins
DARKARENA — +50 XP / +250 coins / Crimson Hit Sparks
```

Промокод можно использовать один раз на игрока. У каждого промокода есть лимит использований.

## Anti-cheat

Система проверяет результат боя перед сохранением:

- слишком быстрый win;
- слишком частая отправка результатов;
- неправильный счёт раундов;
- завышенные rewards;
- результат без Telegram verification;
- попытка отправить бой забаненным игроком.

Подозрительные события записываются в таблицу:

```text
anti_cheat_events
```

## Блокировка игрока

Если игрок забанен, `/api/result` перестанет принимать его результаты.

Поля в таблице `users`:

```text
banned
ban_reason
admin_note
```

## Admin Audit

Каждое admin-действие записывается в:

```text
admin_audit_logs
```

Это нужно, чтобы понимать, кто менял экономику, выдавал награды, банил игроков или создавал промокоды.
